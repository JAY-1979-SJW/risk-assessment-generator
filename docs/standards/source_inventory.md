### 1.1 산안법 시행규칙 제37조 기록 항목 (법정 필수)

규정상 아래 6개 항목을 **반드시** 기록·보존해야 한다:

1. 위험성평가 대상의 유해·위험요인
2. 위험성 결정의 내용
3. 위험성 결정에 따른 조치의 내용
4. (그 밖에) 위험성평가 실시내용 확인을 위한 필요 사항으로 고용노동부장관이 정하는 사항

**보존 기간**: 3년.

### 3.1 공통 5-시트 구조

| 시트 | 역할 |
|------|------|
| 1. 안전보건방침 및 추진목표 | 회사 정보·방침 선언 |
| 2. 위험성평가 조직구성 | 역할·책임·참여자 명단 |
| 3. 위험성 추정 및 결정 | 가능성·중대성 채점 기준표 (3×3 또는 3×3×빈도) |
| 4. 위험성평가 회의 결과 | 회의일시·장소·참석·회의내용 |
| 5. **위험성평가실시** (= **본표**) | hazard 1건 = row 1행. 평가 결과 핵심. |
