## 2. 최우선 수집 대상 (법적 의무, 원본 미확보)

### P1-A: 작업계획서 공란 별지

| 항목 | 내용 |
|------|------|
| 법적 근거 | 산업안전보건기준에 관한 규칙 제38·39·40·42·49·54·57·93·123·139·180·186·191조 등 |
| 현재 상태 | B등급 비원본 3건 (지침서 위주). **법정 공란 별지 없음** |
| 수집 방법 | licbyl API — 산업안전보건기준에 관한 규칙 lawId 추가 수집 |
| 기대 결과 | `data/forms/A_official/licbyl/LAW__별지__작업계획서_*__N_A.hwp(x)` |
| 저장 경로 | `data/forms/A_official/licbyl/` |
| 우선도 | **최우선** (본 엔진 핵심 출력물, P0 승격 후 구조 분석 즉시 착수) |

### P1-B: 안전보건교육 교육일지 (별지 52호의2서식)

| 항목 | 내용 |
|------|------|
| 법적 근거 | 산안법 시행규칙 별지 제52호의2서식 (추정) |
| 현재 상태 | B등급 비원본 1건 (MOEL 의정부지청 배포) — A등급 법정 원본 없음 |
| 수집 방법 | licbyl API — 산안법 시행규칙 별지 52호의2 |
| 기대 결과 | `data/forms/A_official/licbyl/LAW__별지__안전보건교육_교육일지_제52호의2서식__N_A.hwp(x)` |
| 저장 경로 | `data/forms/A_official/licbyl/` |
| 우선도 | **높음** (정기·특별교육 이수 기록 법적 의무) |

### P1-C: 공정안전보고서 관련 서식

| 항목 | 내용 |
|------|------|
| 법적 근거 | 산안법 시행규칙 제50조 (공정안전보고서 제출 의무) |
| 현재 상태 | **0건** — 유해위험방지계획서 별지(제19·22호)는 별도 서식으로 해당 없음 |
| 수집 방법 | licbyl API — 산안법 시행규칙 제50조 관련 별지 |
| 기대 결과 | PSM 관련 법정 별지 서식 |
| 저장 경로 | `data/forms/A_official/licbyl/` |
| 우선도 | **중간** (PSM 대상 사업장만 해당, 일반 건설현장 미해당) |

## 3. 조건부 수집 대상 (현행 B등급으로 임시 운영 가능)

### P1-D: TBM 회의록 양식 공식판

| 항목 | 내용 |
|------|------|
| 현재 상태 | B등급 1건 `KOSHA__TBM__TBM_실행_시나리오_및_회의록_양식__N_A.pdf` |
| 판단 | 법정 서식 없음. 현재 B등급으로 임시 운영 가능 |
| 수집 방법 | KOSHA 기술자료실 재검색 (내부망 접근 시) |
| 조건 | P1-A·B·C 완료 후 착수 |
| 우선도 | **보통** (법정 의무 아님) |

### P1-E: 도급승인 신청서 licbyl 원본 (별지 31·33호)

| 항목 | 내용 |
|------|------|
| 현재 상태 | A등급 4건 (`is_original=N`) — MOEL 정책자료실 배포본 |
| 판단 | 내용·형식이 법정 서식과 동일. 실무 사용 가능 |
| 수집 방법 | licbyl API — 별지 제31호 (도급승인 신청서), 제33호 (도급승인 변경신청서) |
| 조건 | 실무 운용에 문제가 발생할 경우에만 착수 |
| 우선도 | **낮음** (현행 배포본으로 대체 가능) |

## 4. 수집 착수 순서 (권고)

```
P1-A 작업계획서  →  P1-B 교육일지  →  P1-C 공정안전보고서
       ↓ 완료 시 P0 승격 및 구조 분석 즉시 착수
P1-D TBM (조건부)  →  P1-E 도급승인 (조건부)
```

---

## 5. 수집 시 공통 규칙

- 수집 스크립트: `scripts/collect/law_licbyl.py` (licbyl 쿼리 확장)
- 저장 경로: `data/forms/A_official/licbyl/` (자동 분류)
- `source_map.csv` 갱신: `scripts/collect_forms_to_repository.py` 재실행
- 권위 매트릭스 갱신: `docs/standards/form_authority_matrix.md` §2·§5 카운트 갱신
- **대량 수집 금지** — 법정 별지 서식 번호를 특정하여 최소 건수만 수집
