## 6. v1 권위 판정 근거 (레포 내 7개 파일 상세 분석 — 보존)

### 6.1 왜 `경비 xls` 가 A 인가
- 시트 상단에 `KRAS(표준 위험성평가) / (http://kras.kosha.or.kr)` 로 **공단 제공 양식 출처** 명시.
- 17컬럼 구조(위험분류 3분할 + 현재위험성 3분할 + 비고)는 KRAS 웹사이트 다운로드 양식 원형.

### 6.2 왜 `공문기반 v2` 가 "준공식 A" 인가
- 사용처가 **민원/공문 응답용**으로 표준화된 파생이며, 실제 사업장 운영에 가장 자주 쓰이는 실용 레이아웃.
- 본 프로젝트의 `docs/standards/kras_standard_form_v1.md` 가 이 파일을 **공식 준거**로 채택.
- KRAS 원형(A)에서 "비고" 1열만 제거된 경미 파생이라 실질적 권위는 동급.

### 6.3 왜 `표준실시규정.docx` 가 A 인가
- 산안법 제36조 · 시행규칙 제37조 · 고용노동부고시 제2023-19호의 **실시규정 작성 의무**를 충족하는 표준 문서.
- 여러 xlsx 서식이 "표준실시규정 제3조·제4조 참조" 로 조항을 직접 인용 → xlsx 조직구성 시트의 **근거 문서**.
- 다만 "본표 템플릿"은 아니므로 본표 구현 대상 서식은 아니다.

### 6.4 왜 `표준양식_테스트.xlsx` 가 B 인가
- 단일 "유해위험요인" 컬럼으로 축약 + "번호" 컬럼 추가는 KRAS 원형에 없는 변형.
- 단, `kras_standard_form_v1.md` 가 "KRAS 축약 12컬럼 (Compact)" 을 v1 수용 대상으로 명시 → 표준의 **하위 세트로 인정**.

### 6.5 왜 `최종/test` 가 C 인가
- 레이아웃은 공문기반 v2 와 동일하나, 본문 데이터가 개발 중 기록/최소 샘플 수준.
- 레이아웃 기준 원본으로 사용할 이유가 없음 (v2 가 상위).
