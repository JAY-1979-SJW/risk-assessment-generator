**작성일**: 2026-04-24  
**기준**: legal_form_implementation_priority.md + legal_form_gap_matrix.md  
**상태**: 이 문서에 정의된 순서는 카탈로그 기준으로 고정됨. 변경 시 본 문서 개정 필요.

## 우선순위 원칙

| 레벨 | 기준 |
|------|------|
| **P0** | 이미 완성됨 — 유지·안정화만 필요 |
| **P1** | 법정 의무 + 전 업종/현장 공통 + builder 난이도 낮음 (workplan 패턴 재사용) |
| **P2** | 법정 의무 + 적용 범위 넓음 + 별지 수집 완료 또는 중난이도 |
| **P3** | 법정 의무이나 특정 사업장 한정 또는 별지 미수집 + 높은 복잡도 |
| **P4** | 실무 필요이나 법정 강제 약함 또는 대형/전문 사업장 한정 |

## P0 — 완성 (3건, 유지·안정화)

### P0-1: 위험성평가표 (`risk_assessment`)

- **현재 상태**: DONE (커밋 548ceee)
- **이유**: 법정 필수 1순위, 전 사업장 대상, KRAS DB 연계 핵심
- **선행 조건**: 없음 (완료)
- **유지 과제**: kras_standard_form_v1.md 스키마 변경 시 builder 동기화, 교육일지·작업계획서와 통합 테스트

### P0-2: 안전보건교육일지 (`education_log`)

- **현재 상태**: DONE (v1.1)
- **이유**: 법정 필수, 전 사업장(1인 이상) 대상, 이미 API 운영 중
- **유지 과제**: 별지 52호의2 원본 수집 시 레이아웃 갱신 검토

### P0-3: 굴착 작업계획서 (`excavation_workplan`)

- **현재 상태**: DONE (v1.0)
- **이유**: 건설 현장 기본 서류, workplan_builder 패턴의 기준
- **유지 과제**: 커밋 cc0c777에서 추가된 필드(법정 준수 항목) 통합 테스트. 출입통제·유도자 필드 누락 여부 재확인

## P1 — 단기 구현 (2건, 1~2개월)

### P1-1: 차량계 건설기계 작업계획서 (WP-002)

- **이유**: 건설 현장 거의 전수 적용, 근로감독 1순위 확인 서류, 미작성 시 과태료 500만원
- **선행 조건**: WP-001(`workplan_builder.py`) 안정화 완료 (P0-3) ✅
- **구현 방법**:
  1. `workplan_builder.py`에서 `LEGAL_LABELS` 배열 교체 (4개 법정 필수 항목)
  2. `required_fields`: `machine_type`, `machine_capacity`, `travel_route`, `work_method`
  3. `optional_fields`: `site_name`, `work_location`, `work_date`, `ground_survey`, `guide_worker_plan`, `access_control`, `work_radius`, `emergency_measure`, `operator_license`, `supervisor`, `sign_date`
  4. `form_registry.py`에 `FormSpec` 1행 추가 → form_type: `construction_machine_workplan`
  5. 추가 API 코드 변경 없음
- **예상 공수**: 1~2일

### P1-2: 차량계 하역운반기계 작업계획서 (WP-003)

- **이유**: 지게차 사용 사업장 전체(제조·물류·건설 공통), 근로감독 1순위, 미작성 과태료 500만원
- **선행 조건**: P1-1 구현 후 패턴 검증 완료
- **구현 방법**:
  1. P1-1과 동일 패턴, `machine_capacity` → `machine_max_load` 교체
  2. `required_fields`: `machine_type`, `machine_max_load`, `travel_route`, `work_method`
  3. `optional_fields`: `site_name`, `work_location`, `work_date`, `pedestrian_separation`, `guide_worker_plan`, `access_control`, `speed_limit`, `refuel_location`, `pre_check_items`, `emergency_measure`, `operator_license`, `supervisor`, `sign_date`
  4. form_type: `material_handling_workplan`
- **예상 공수**: 1일 (P1-1 검증 후 단순 파라미터 변경)

## P2 — 단기 구현 (4건, 2~3개월)

### P2-1: 산업재해조사표 (ACC-001)

- **이유**: 전 사업장, 재해 발생 1개월 내 제출 의무, 미제출 과태료 1,500만원 — 실제 사용 빈도는 낮으나 제출 기한이 있어 디지털 작성 수요 큼
- **선행 조건**: 별지 제30호서식 수집 완료 ✅ (licbyl A등급)
- **구현 방법**:
  1. `data/forms/A_official/licbyl/` 내 별지 30호 HWP 파일 레이아웃 분석
  2. `accident_report_builder.py` 신규 작성 (openpyxl로 법정 레이아웃 재현)
  3. required_fields 24개 (LAW 항목 전체)
  4. FORB: `victim_ssn` 필드 미정의 — 주민번호 미수집
  5. form_type: `accident_report`
- **예상 공수**: 2~4일

### P2-2: 산업안전보건위원회 회의록 (MTG-001)

- **이유**: 100인 이상 사업장 전수 적용, 분기별 반복 발행, 별지 수집 완료
- **선행 조건**: 별표 제2의2 수집 완료 ✅ (licbyl A등급)
- **구현 방법**:
  1. 별표 2의2 레이아웃 재현 (단순 표 구조)
  2. `committee_minutes_builder.py` 신규 작성
  3. required_fields 8개 (회의일시, 장소, 위원장, 근로자위원, 사용자위원, 의안, 의결, 서명)
  4. 회의 차수 자동 순번 (`meeting_no`)
  5. form_type: `safety_committee_minutes`
- **예상 공수**: 1~2일 (단순 표 형태)

### P2-3: 타워크레인 작업계획서 (WP-004)

- **이유**: 중대재해 다발 고위험, 중대재해처벌법 연계, 타워크레인 현장 = 고위험 대규모 공사
- **선행 조건**: P1-1 패턴 안정화
- **구현 방법**: workplan_builder 패턴 + 설치·해체 단계 섹션 추가
- **예상 공수**: 1~2일

### P2-4: 중량물 취급 작업계획서 (WP-005)

- **이유**: 제조·물류·건설 광범위, 중량물 취급 사고 다발
- **선행 조건**: P1-1 패턴 안정화
- **구현 방법**: workplan_builder 패턴 재사용
- **예상 공수**: 1일

## P3 — 중기 구현 (6건, 3~6개월)

### P3-1: 건설공사 유해위험방지계획서 (HRP-002)

- **이유**: 공사금액 50억 이상 필수, 미제출 시 착공 금지 — 법적 파급력 최고 수준
- **선행 조건**: 별지 17·22·103호 수집 완료 ✅. 구현 전 서식 구조 분석 선행 필수
- **구현 특성**: 첨부서류 목록(별표 10) 방대 → builder는 본표만, 첨부 안내는 별도 처리
- **예상 공수**: 5~10일 (레이아웃 복잡)

### P3-2: 제조업 유해위험방지계획서 (HRP-001)

- **이유**: 일정 규모 이상 제조업, 미심사 시 설치·운전 금지
- **선행 조건**: 별지 16·18·19호 수집 완료 ✅
- **예상 공수**: 5~10일

### P3-3: 유해·위험작업 도급승인 신청서 (CON-001)

- **이유**: 유해위험 도급 시 착공 전 제출 의무, 별지 수집 완료
- **선행 조건**: 별지 31·32·33·34호 수집 완료 ✅
- **구현 방법**: 신청서 형태 단순 — builder 난이도 낮음
- **예상 공수**: 2~3일

### P3-4: 터널 굴착 작업계획서 (WP-006)

- **이유**: 터널공사 법정 필수, 전문 항목 多
- **선행 조건**: workplan_builder 패턴 안정화 + 터널 굴착 전문 항목 정의
- **예상 공수**: 2~3일

### P3-5: 건축물 해체 작업계획서 (WP-007)

- **이유**: 해체공사 법정 필수
- **선행 조건**: workplan_builder 패턴
- **예상 공수**: 1~2일

### P3-6: 노사협의체 회의록 (MTG-002)

- **이유**: 건설 120억 이상 도급 한정
- **선행 조건**: 별표 31의2 licbyl 수집 선행 (현재 미수집)
- **예상 공수**: 1~2일 (수집 완료 후)
