
## 2. 문서별 UI 상태 정의

각 문서 카드에 표시할 4가지 상태 배지:

| 배지 | 색상 | 의미 |
|------|------|------|
| 🟢 생성 가능 | 초록 | DONE — builder + API 완료, 즉시 Excel 생성 |
| 🟡 준비중 | 노랑 | TODO — 개발 예정, 예상 완료 시기 표시 |
| 🔵 원본 보기 | 파랑 | source_map A등급 별지 수집됨 — HWP 다운로드 가능 |
| ⚫ 법적 근거 | 회색 | 모든 문서 — 법령 조문 팝업 링크 |

## 5. 상태 배지 로직

```python
def get_badge(doc):
    if doc.current_status == "DONE":
        return "생성 가능"          # 🟢
    elif doc.current_status == "TODO":
        return f"준비중 ({doc.priority})"  # 🟡
    elif doc.current_status == "EXCLUDED":
        return None                 # 메뉴에 미표시

def get_source_button(doc):
    if doc.source_file_exists == "Y" and doc.source_authority == "A_OFFICIAL":
        return "원본 보기"          # 🔵
    return None
```

## 6. 메뉴 노출 정책

| 정책 | 내용 |
|------|------|
| DONE 서류 | 메뉴 최상단, 생성 가능 배지 |
| TODO P1·P2 서류 | 메뉴 표시, 준비중 배지 + 예상 시기 표시 |
| TODO P3·P4 서류 | "더 보기" 섹션으로 접어두기, 클릭 시 확장 |
| EXCLUDED 서류 | 메뉴 미표시 (PSM, 안전보건관리규정, TBM 일지) |
| A_OFFICIAL 별지 수집 서류 | 원본 보기 버튼 항상 표시 (생성 가능 여부 무관) |
