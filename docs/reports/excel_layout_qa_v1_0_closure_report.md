## 🎯 A4 출력 기준 (최종)

### 인쇄 설정
```
ws.page_setup.orientation = "portrait" or "landscape"
ws.page_setup.paperSize = 9 (A4)
ws.page_setup.fitToWidth = 1
ws.page_setup.fitToHeight = 0 (다중페이지 자연 분할)
ws.page_margins = PageMargins(left=0.5, right=0.5, top=0.75, bottom=0.75)
```

### 너비 기준 (Portrait)
| 기준 | 값 |
|-----|-----|
| A4 용지 폭 | 8.27in |
| 좌우 여백 | 0.5in × 2 = 1.0in |
| 출력 가능 폭 | 7.27in |
| FAIL 기준 | > 1.40배 = 10.18in |
| WARN 기준 | > 1.60배 = 11.63in |
| PASS 기준 | ≤ 1.60배 = 11.63in |

### 너비 기준 (Landscape)
| 기준 | 값 |
|-----|-----|
| A4 용지 폭 (회전) | 11.69in |
| 좌우 여백 | 0.5in × 2 = 1.0in |
| 출력 가능 폭 | 10.69in (권고), 11.09in (fitToWidth=1) |
| WARN 기준 | > 1.60배 × 10.69in = 17.10in |
| PASS 기준 | ≤ 1.60배 × 11.09in = 17.74in |

### 행 높이 기준
- **최소 높이:** 15pt (기본값)
- **표준 높이:** 20pt
- **서명란:** ≥18pt (공식 사인 공간)
- **섹션 헤더:** 18~22pt
- **제목:** 28pt 이상

### 행 역할별 적용
| 요소 | 최소 높이 | 표준 높이 | 비고 |
|-----|---------|---------|------|
| 문서 제목 | 26pt | 28pt+ | FONT_TITLE |
| 섹션 헤더 | 18pt | 20~22pt | 법령/규정 참조 |
| 메타정보 (레이블) | 18pt | 20pt | 사업장명, 현장명 등 |
| 테이블 헤더 | 20pt | 20pt | 컬럼 이름 |
| 데이터 행 | 15pt | 20~28pt | 행 높이는 콘텐츠 양에 따라 |
| 서명란 | 18pt+ | 28~36pt | **공식 사인 공간 보장** |
| 주석/안내 | 14pt | 16pt | 작은 텍스트 |

### wrap_text 설정
- **필수:** 20자 이상 텍스트 셀 (자동 줄바꿈)
- **권장:** 레이블 셀 (한글 2줄 이상)
- **제외:** 날짜, 숫자, 짧은 선택지

### 반복 헤더 (print_title_rows)
- **표형 문서:** 반복 헤더 행 설정 (테이블 상단 고정)
- **비표형 문서:** 제목/부제 행만 반복 ("1:2")
- **예:**
  - 체크리스트: "1:6" (제목, 부제, 안내, 공백, 헤더 행)
  - 점검표: "1:4" (제목, 부제, 법령, 헤더)
  - 회의록: "1:2" (제목, 부제만)

## ⚠️ ACCEPTED_WARN

### risk_assessment (위험성평가표 본표)

**WARN 내용:**
```
열 너비 합계 28.77in > 출력폭 11.09in (259%)
fitToWidth=1 적용 시 과도한 축소 가능
```

**기술적 배경:**
- **컬럼 수:** 17개 (번호, 공정명, 세부작업명, 위험분류 대/중, 유해위험요인, 법적기준, 현재조치, 평가척도, 가능성, 중대성, 위험성, 감소대책, 개선후 위험성, 예정일, 완료일, 담당자)
- **원본 너비:** 262 unit × 0.1098 = 28.77in
- **A4 landscape 출력폭:** 11.09in (fitToWidth=1 적용 시)
- **축소율:** 259% (28.77 ÷ 11.09)

**허용 사유:**
1. **법령 기준 컬럼:** 모든 컬럼이 산업안전보건 표준 평가 항목
   - 「산업안전보건기준에 관한 규칙」 제627조 (위험성 수준의 결정) 참조
   - 삭제 불가능
2. **실무 필요성:** 17개 항목 모두 위험성평가 의무 교육 대상
3. **구조적 한계:** 컬럼 병합/숨김 불가
4. **현업 수용:** fitToWidth=1 설정으로 자동 축소 (읽기 가능 수준)

**조치:**
- ✅ fitToWidth=1 적용 (자동 축소)
- ✅ landscape 방향 유지 (넓은 폭 활용)
- ✅ print_area 명시 (인쇄 범위 고정)
- ✅ wrap_text 적용 (모든 텍스트 셀)

**향후 개선안 (V2.0):**
| 매체 | 전략 | 상세 |
|-----|-----|------|
| **웹 화면** | 가로 스크롤 / 섹션 분할 | 위험분류/유해요인, 평가/척도, 조치/일정 탭 분리 |
| **PDF 출력** | A3 용지 또는 분할 출력 | 좌측(컬럼 1-8) + 우측(컬럼 9-17) 2장 |
| **Excel** | 명시적 축소율 설정 | 80% ~ 85% 고정값 (사용자 선택) |

**결론:**
- WARN은 **허용 가능한 수준**
- 컬럼 구조 유지 필수
- 현 설정으로 충분히 실무 운영 가능
- V2.0 웹 화면 개발 시 위의 전략 검토 필요
