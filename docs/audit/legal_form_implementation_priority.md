## 구현 전략 권고

### 작업계획서 시리즈 (P1~P3)

`workplan_builder.py` 패턴(굴착) 재사용:
1. `LEGAL_LABELS` 배열에 각 작업계획서 법정 필수 항목 교체
2. `required_fields` / `optional_fields` 재정의
3. `form_registry.py`에 신규 `FormSpec` 추가
4. 추가 API 코드 변경 없음 (registry 확장만으로 자동 지원)

예상 공수: 서류당 1~2일 (레이아웃 설계 포함)

### 별지 수집 서류 (산업재해조사표, 위원회 회의록 등)

1. `data/forms/A_official/`에서 해당 별지 HWP 파일 참조
2. 법정 서식 레이아웃 재현 (openpyxl)
3. required_fields는 법령 조항에서 직접 추출

예상 공수: 서류당 2~4일 (레이아웃 재현 포함)

### 유해위험방지계획서 (P3)

- 첨부서류 목록(별표 10)이 방대 → 작성 보조 도구 수준으로 범위 조정 필요
- 별지 수집됨이므로 구현 착수 전 서식 구조 분석 선행
