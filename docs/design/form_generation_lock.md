## 2. 작업계획서

### 2.1 v1 생성 여부: **YES — 구현 대상**

| 항목 | 판단 |
|------|------|
| Excel 생성 가능성 | **가능** — 작업 유형별 단일 시트 구조 |
| 법정 항목 완전성 | **가능** — 조문에서 항목 100% 추출 완료 |
| 자동화 기여도 | **낮음** — AUTO 가능 필드 3개 (작성일, 순번, 작업유형 드롭다운). 나머지 USER 입력 |
| 원본 서식 의존도 | **없음** — 법령이 형식 지정 없음. 조문 항목 충족으로 합법 서식 생성 가능 |

### 2.2 확정 필드 (v1 구현 범위)

**공통 메타 + 하나의 작업 유형(굴착)으로 v1 첫 구현 후 다른 유형 확장:**

| 그룹 | 필드 수 | 비고 |
|------|---------|------|
| 메타 영역 | 8 | site_name, work_type, work_location, work_date_start, work_date_end, prepared_by, prepared_date, supervisor |
| 공통 법정 항목 | 2 | work_method, emergency_measure |
| 굴착 법정 항목 | 5 | excavation_method, earth_retaining, excavation_machine, soil_disposal, water_disposal |
| 안전조치 반복행 | 4 | step_no, step_description, hazard_point, safety_action |
| 서명 영역 | 3 | sign_preparer, sign_supervisor, sign_date |
| **합계** | **22** | |

### 2.3 구현 범위 제한

| 항목 | 범위 |
|------|------|
| v1 구현 유형 | **굴착 작업** 1종 (법정 기재 항목이 가장 많아 스키마 검증에 유리) |
| v1.1 확장 유형 | 차량계 건설기계, 차량계 하역운반기계 |
| v1.2 확장 유형 | 터널, 해체, 중량물 |
| 작업유형 드롭다운 | 제38조 각호 전체 목록 표시 (단, 미구현 유형은 "준비 중" 표시) |

### 2.4 출력 형식

- **형식**: Excel (.xlsx), 단일 시트
- **시트명**: `작업계획서`
- **생성 방식**: `excel_builder` 신규 모듈 (`workplan_builder.py`)
- **레이아웃**: 메타 블록 상단 → 법정 항목 블록 중단 → 안전조치 테이블 하단 → 서명 최하단

## 4. 공정안전보고서 (PSM)

### 4.1 v1 생성 여부: **NO — v1 범위 외**

| 항목 | 판단 |
|------|------|
| Excel 생성 가능성 | **불가** — 4대 구성요소가 각각 독립 문서(도면·표·서술 혼합). 단일 Excel로 불완전 |
| 법정 항목 완전성 | **높음** — 별표 13 항목 추출 완료. 그러나 도면(P&ID·배치도)은 Excel 생성 범위 외 |
| 자동화 기여도 | **매우 낮음** — 전 항목이 전문 엔지니어링 입력 필요 |
| 원본 서식 의존도 | **높음** — KOSHA 심사 양식 구조를 따라야 하나 미확보 |

### 4.2 v1 배제 사유

1. **패키지 문서 구조**: Excel 단일 파일로 생성 불가. 최소 4개 섹션 × 각각 도면·표·서술 혼합.
2. **전문 입력 의존**: 위험성 평가(HAZOP 등)는 별도 전문 분석 결과를 전제. 자동화 기여 없음.
3. **KOSHA 심사 요구사항**: 심사 통과 요건에서 세부 양식 요구가 있으나 미확보.
4. **PSM 적용 사업장 한정**: 유해위험물질 규정량 이상 사업장에만 적용 → 본 앱 주요 사용자(중소 사업장)와 부합도 낮음.

### 4.3 향후 구현 방향 (v2 이후)

| 단계 | 내용 |
|------|------|
| v2 착수 조건 | KOSHA PSM 심사 양식 원본 확보 + PSM 대상 사업장 사용자 요구 확인 |
| v2 구현 범위 | PSM-1 (공정안전자료) 중 설비목록·물질목록 테이블만 Excel 출력 |
| v2.1 | PSM-2 위험성평가서 (HAZOP 워크시트 표) |
| v3 | PSM 패키지 전체 (워드/PDF 복합 출력) |
