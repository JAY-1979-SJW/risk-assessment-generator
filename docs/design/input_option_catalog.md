# Input Option Catalog (v2.0)

**대상 API**: `POST /api/v1/risk-assessment/build`
**시행일**: 2026-04-23
**범위**: 10개 work_type와 상식적으로 연결 가능한 범위만 1차 수록.

---

## 정책 요약

| 상황 | 처리 |
|------|------|
| 허용값 외 입력 | 400 `INVALID_INPUT_OPTION` |
| 배열 내 빈 문자열 | 400 `INVALID_INPUT_OPTION` |
| 배열 내 중복값 | 자동 dedupe (순서 보존, 에러 없음) |
| 빈 배열 `[]` | 해당 필드 무시 (v1 동작) |
| `null` 또는 필드 미존재 | 해당 필드 무시 (v1 동작) |

---

## 1. equipment (장비) — 8개

| 값 | 설명 |
|----|------|
| `사다리` | 이동식 A자/일자 사다리. 고소작업·전기작업·용접작업 등에서 추락위험 상향. |
| `이동식비계` | 바퀴 달린 이동식 작업대. 전도/추락 위험. |
| `고소작업대` | 차량탑재형·자주식 고소작업대(MEWP). 끼임·전도·낙하 위험. |
| `전동공구` | 드릴·그라인더 등 전원공구. 감전·절단·비산 위험. |
| `절단기` | 휴대용 그라인더·컷오프 쏘. 절단·비산·화재 위험. |
| `용접기` | 아크/가스 용접기. 화재·감전·흄 위험. |
| `크레인` | 이동식·타워·고정식 크레인. 낙하·충돌 위험. |
| `굴착기` | 백호/굴삭기. 매몰·협착·전도 위험. |

## 2. location (장소) — 8개

| 값 | 설명 |
|----|------|
| `옥상` | 건물 지붕면. 추락 위험 1순위. |
| `실내` | 실내 작업 공간. 환기/화재/유해가스 누적 위험. |
| `지하` | 지하층·지하실. 산소결핍·질식 가능. |
| `맨홀` | 맨홀/집수정. 밀폐공간, 질식·유독가스. |
| `협소구간` | 좁은 통로/구획. 협착·탈출지연. |
| `외부` | 외부/옥외 작업. 우천·낙뢰·강풍 환경 영향. |
| `천장부` | 천장 설치·배관 작업. 추락·낙하물 위험. |
| `PIT` | 피트/점검구. 밀폐공간 준하는 환경, 질식·화재 위험. |

## 3. conditions (추가조건) — 10개

| 값 | 설명 |
|----|------|
| `활선근접` | 충전전로 인근. 감전 위험 극대화. |
| `화기작업` | 용접·가스절단·그라인딩. 화재·폭발 위험. |
| `중량물취급` | 25kg 이상 중량물. 근골격계·낙하 위험. |
| `협업작업` | 2인 이상 동시 수행. 신호 오류·충돌 위험. |
| `야간작업` | 일출 전/일몰 후. 시인성·피로 위험. |
| `우천` | 강우 중 작업. 미끄러짐·감전·낙뢰 위험. |
| `밀폐공간` | 산소결핍·유해가스 가능 공간. |
| `추락위험` | 2m 이상 고소. 추락재해 위험. |
| `비산분진` | 분진·분체 발생. 호흡기/시야 위험. |
| `소음발생` | 85dB 초과 소음 환경. 청력 위험. |

## 확장 원칙

- 신규 값 추가는 **카탈로그 + rulebook + schema 동시** 갱신 필요.
- 값 제거는 하위호환 위반 → 실질상 금지 (deprecate 후 다음 메이저에서만).
- 유사어/자연어(예: "지붕", "아크 용접")는 v1~v2 범위에서 **수용하지 않음**. AI 정규화 단계에서 처리 예정.
