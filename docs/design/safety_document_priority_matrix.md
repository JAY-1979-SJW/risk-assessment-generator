## 2. 우선순위 판정 기준

각 항목을 0~3점으로 평가, 합산 점수 순으로 정렬.

| 기준 | 설명 | 배점 |
|------|------|------|
| **법적 의무 강도** | 법령 위반 시 과태료·처벌 여부 | 0~3 |
| **현장 빈도** | 건설·제조 현장에서 작성 빈도 (월 기준) | 0~3 |
| **미구현 위험도** | 미구현 시 법적 누락 위험·현장 혼란 수준 | 0~3 |
| **구현 복잡도** (역산) | 낮을수록 높은 점수 (복잡하면 감점) | 0~3 |
