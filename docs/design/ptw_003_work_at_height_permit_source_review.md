- 비계·고소작업대·사다리·작업발판 중복 서식 범위 구분

## 3. 법정 별지 서식 존재 여부

**결론: 법정 별지 서식 미확인**

산업안전보건기준에 관한 규칙에 "고소작업 허가서" 또는 "추락위험작업 허가서"라는 이름의 **법정 별지 서식은 규정되지 않는다.**

법령이 요구하는 것은 다음과 같다.

- 제42조~제45조: 추락 위험 장소에서의 설비 설치 및 안전조치 의무 (작업발판, 안전난간, 추락방호망, 안전대 부착설비 등)
- 이 조항들은 **설비 설치 및 안전조치 의무**이며, 허가서 발행 의무가 아니다.
- "허가서" 또는 "별지 서식"이라는 단어는 제42조~제45조 본문에 존재하지 않는다.

산업안전보건법 시행규칙 별표·별지에서도 고소작업 관련 허가서 서식은 확인되지 않았다.

**검색한 경로:**
- 산업안전보건기준에 관한 규칙 전문 (lbox.kr, law.go.kr)
- 국가법령정보센터 별표·서식 목록
- 산업안전보건법 시행규칙 별지 목록

## 4. 적용 조항 정리

### 4.1 산업안전보건기준에 관한 규칙 — 추락 관련 핵심 조항

| 조항 | 제목 | PTW-003 관련성 | 의무 성격 |
|------|------|----------------|-----------|
| 제37조 | 악천후 및 강풍 시 작업 중지 | 기상조건 기준 | 작업 중지 의무 |
| 제42조 | 추락의 방지 | 허가서의 핵심 법령 근거 | 설비 설치·안전조치 의무 |
| 제43조 | 개구부 등의 방호조치 | 개구부 방호 점검 근거 | 설비 설치 의무 |
| 제44조 | 안전대의 부착설비 등 | 안전대 부착설비 점검 근거 | 설비 설치 의무 |
| 제45조 | 지붕 위에서의 위험 방지 | 지붕 작업 시 적용 | 안전조치 의무 |
| 제57조 이하 | 비계 등의 조립·해체·변경 | 비계 위 작업 관리감독자 | 관리감독자 배치 의무 |
| 제69조 이하 | 작업발판 기준 | 작업발판 폭·틈새 기준 | 설비 기준 (조항번호 NEEDS_VERIFICATION) |
| 제86조 이하 | 고소작업대 설치 등의 조치 | 고소작업대 설치 기준 | 설비 설치 의무 |

### 4.2 제42조 — 추락의 방지 의무 내용

1. 비계 조립 등의 방법으로 작업발판 설치
2. 작업발판 설치 곤란 시 추락방호망 설치
3. 추락방호망 설치 곤란 시 근로자에게 안전대 착용
   - 안전대 착용 시 제44조에 따른 부착설비 설치 병행

**의무 단계:** 작업발판 → 추락방호망 → 안전대 (단계적 적용)

### 4.3 제43조 — 개구부 등의 방호조치 의무 내용

- 작업발판 및 통로 끝, 개구부에 안전난간·울타리·수직형 추락방망·덮개 설치
- 충분한 강도의 방호조치 요건

### 4.4 제44조 — 안전대의 부착설비 등

- 높이 2m 이상 장소에서 안전대 착용 시 부착설비 설치 의무
- 부착설비는 충분한 강도를 가진 구조로 설치

### 4.5 제37조 — 악천후 시 작업 중지 기준

- 강풍(10분 평균 풍속 10m/s 이상): 작업 중지 권장 (비계·고소작업대 작업)
- 강우·강설: 작업 중지 고려

### 4.6 법령상 의무 성격 종합

| 법령 의무 | 서식 요건인가? |
|----------|--------------|
| 제42조 작업발판·추락방호망·안전대 조치 | 서식 아님. 설비/조치 의무 |
| 제43조 개구부 방호조치 | 서식 아님. 설비 설치 의무 |
| 제44조 안전대 부착설비 설치 | 서식 아님. 설비 설치 의무 |
| 제57조 비계 조립·해체 관리감독자 배치 | 서식 아님. 인적 배치 의무 |
| 제86조 이하 고소작업대 설치 기준 | 서식 아님. 설비 기준 의무 |

## 8. PTW-003 구현 기준 확정

### 8.1 법정 별지 서식 여부

**PTW-003은 법정 별지 서식이 아니다.**  
산업안전보건기준에 관한 규칙은 고소작업 허가서 서식을 별지로 지정하지 않는다.  
법정 의무는 제42조~제44조의 **설비 설치 및 안전조치 의무**다.

### 8.2 구현 방식

법정 서식 아님 → **자체 표준서식으로 구현 가능**  
근거: 제42조~제44조 안전조치 이행 확인 양식 + KOSHA GUIDE P-94-2021 참고

### 8.3 고소작업 허가 필요 조건

- **높이 2m 이상** 추락위험 작업
- 비계, 고소작업대, 사다리, 작업발판 사용 작업
- 원청 안전 관리 규정에서 허가 요구하는 경우

### 8.4 비계·고소작업대·사다리·작업발판 구분 원칙

| 유형 | PTW-003 역할 | 세부 점검 서식 |
|------|-------------|--------------|
| 비계 | 비계 위 작업 허가 여부 확인 | CL-001 (비계·동바리 설치 점검표) |
| 고소작업대 (스카이차·붐리프트·시저리프트) | 고소작업대 사용 허가·작업 전 확인 | CL-003 (건설장비 일일 사전점검표) / EQ-005 |
| 사다리 | 사다리 사용 제한 기준 확인 | PTW-003 섹션 내 간략 확인 |
| 작업발판 (독립형) | 작업발판 설치 상태 확인 | PTW-003 섹션 내 간략 확인 |

**핵심 원칙:**
- PTW-003은 **고소작업 허가 여부 판단 및 작업 전 안전조치 이행 확인** 문서
- CL-001(비계), CL-003(장비)은 **설치·사전 점검 전문 문서** — PTW-003이 대체하지 않음
- 고소작업대는 CL-003 또는 EQ-005와 병행 사용

### 8.5 안전대·추락방지조치 반영 기준

- 제42조 단계적 의무: ① 작업발판 설치 → ② 추락방호망 설치 → ③ 안전대 착용 순
- builder에서 선택 구조로 구현: "작업발판 설치 가능 여부" → "추락방호망 설치 여부" → "안전대 착용 확인"
- 안전대 착용 시 제44조 부착설비 설치 여부 연동 확인 항목 필수

### 8.6 사진 증빙 정책

| 항목 | 정책 |
|------|------|
| 법적 의무 여부 | 아님 |
| 권장 정책 | OPTIONAL |
| 권장 항목 | 작업 전 추락위험 장소, 안전설비 설치 상태, 안전대 착용 상태, 고소작업대 아웃트리거 전개 상태 |

PTW-002와 동일한 정책 적용: "사진은 법정 필수 고정항목이 아니라 점검 대응을 위한 권장 증빙으로 관리한다."

### 8.7 evidence_status 초기값

- `legal_status`: `practical` (법정 별지 서식 없음. 원청 요구 실무 필수)
- `evidence_status`: `NEEDS_VERIFICATION`
  - 법령 조항 확인: 조항명·의무 성격 확인. 조항 원문 API 미수집 — NEEDS_VERIFICATION
  - KOSHA GUIDE 원문 직접 확인: PDF 텍스트 추출 불가 — NEEDS_VERIFICATION
  - (참고: PTW-002는 일부 조항 내용 확인으로 PARTIAL_VERIFIED 시작. PTW-003은 의존 조항 원문 수집 없음으로 NEEDS_VERIFICATION 시작)

## 11. 다음 builder 구현 지시 시 주의사항

1. **PTW-003은 법정 별지 서식 아님** — `legal_status: practical` 유지. 자체 표준서식으로 구현.
2. **evidence_status: NEEDS_VERIFICATION으로 시작** — 법령 조항 원문 미수집 상태.
3. **섹션 6·7은 조건부 활성화** — 고소작업 유형(섹션 3) 선택에 따라 비계 확인(섹션 6), 고소작업대 확인(섹션 7) 섹션을 선택적으로 활성화.
4. **CL-001/CL-003 대체 금지** — PTW-003은 허가·간략 확인 문서. 비계 설치 점검(CL-001), 장비 점검(CL-003)을 대체하지 않는다는 안내 문구 포함.
5. **안전대 착용 시 부착설비 연동** — 제44조 근거. 안전대 착용 항목 선택 시 부착설비 설치 여부 확인 항목이 자동 활성화되도록 구현.
6. **기상조건 섹션 필수** — 제37조 악천후 작업 중지 기준. 고소작업은 기상조건에 민감하므로 풍속·우천 확인 항목 포함.
7. **KOSHA GUIDE P-94-2021**을 구현 참고 기준으로 사용. 원문 PDF 텍스트 추출 불가로 허가 절차(신청→확인→승인→현장게시) 구조만 참고.
8. **PTW-002와 동일한 사진 정책** — OPTIONAL, 법정 필수 아님 명시.
9. **고소작업대 섹션(7)**은 EQ-005 구현 후 연계 정책 재검토 필요.
