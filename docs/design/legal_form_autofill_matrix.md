**입력**: legal_form_field_inventory.md + legal_form_field_classification.md  
**목적**: 각 항목의 자동 채움 소스를 판정하여 builder 입력 최소화 및 사용자 입력 부담 경감

## 자동 채움 소스 정의

| 소스 코드 | 소스명 | 설명 |
|----------|--------|------|
| `SRC_SITE` | 사업장 마스터 | 사업장명, 주소, 업종, 대표자, 상시근로자수 등 |
| `SRC_RISK` | 위험성평가 결과 | 유해위험요인, 현재조치, 위험성, 감소대책 등 |
| `SRC_SYS` | 시스템 자동 | 작성일시, 일련번호, 차수 등 |
| `SRC_HIST` | 이전 서류 이력 | 이전 작업계획서, 이전 교육일지 등 |
| `SRC_USR` | 사용자 직접 입력 | 현장 특수 정보로 자동 채움 불가 |

## 자동 채움 가능성 종합 요약

### 문서별 자동 채움 비율

| 문서 | 전체 항목 | 자동 채움 가능 | 비율 | 비고 |
|------|---------|------------|------|------|
| 위험성평가표 | 23 | 9 | 39% | KRAS DB 연계 시 상승 가능 |
| 안전보건교육일지 | 18 | 10 | 56% | 이전 교육 이력 활용 시 |
| 굴착 작업계획서 | 17 | 8 | 47% | 이전 계획서 재사용 시 |
| 차량계 건설기계 작업계획서 | 17 | 11 | **65%** | 표준항목 많아 자동화 유리 |
| 차량계 하역운반기계 작업계획서 | 16 | 10 | **63%** | 표준항목 많아 자동화 유리 |
| 산업재해조사표 | 24 | 5 | **21%** | 재해 발생 후 기술 특성상 최저 |
| 안전보건위원회 회의록 | 11 | 7 | **64%** | 위원회 마스터 활용 시 |

| 유해위험방지계획서 | 15 | 9 | 60% | 위험성평가 연계 필수 |

### 자동 채움 소스별 기여 항목 수

| 소스 | 기여 항목 수 | 주요 항목 |
|------|------------|---------|
| `SRC_SITE` (사업장 마스터) | 42 | 사업장명, 주소, 업종, 대표자, 담당자 |
| `SRC_RISK` (위험성평가) | 18 | 위험요인, 감소대책, 위험성 수준 |

### 사용자 입력 필수 — 절대 자동 채움 불가 항목

|------|------|
| 재해 발생경위 | 재해 후 현장 조사 결과 |

| 실제 운행경로 | 현장 배치 변경 수시 발생 |

| 위험성 수준 최종 판단 | 평가자 직접 판단 (가능성·중대성) |
| 의결사항 | 회의 결과 |
| 인접 구조물 영향 | 현장 특수 조건 |
| 운전자 면허번호 | 투입 인원 변동 |

### 위험성평가 ↔ 작업계획서 연계 자동 채움 항목

위험성평가 결과를 작업계획서 생성 시 자동으로 활용 가능한 항목:

| 위험성평가 필드 | → | 작업계획서 필드 |
|--------------|---|--------------|
| `hazard` | → | `safety_steps[].hazard` |
| `control_measures` | → | `safety_steps[].safety_measure` |
| `control_measures` | → | `emergency_measure` (일부) |
| `process` | → | `work_method` (공정 참조) |
| `legal_basis` | → | 유해위험방지계획서 위험요인 근거 |
