# 작업계획서 실제 레이아웃 구조 분석 — 2단계

**작성일**: 2026-04-24  
**분석 대상**: 실제 표 구조가 확인된 서식 2건 + 법령 지침 1건

---

## 1. KOSHA C-102-2023 부록 — 중량물 취급 작업계획서 양식

### 1.1 표지 (p.9)

```
중량물 취급 작업계획서 (이동식 크레인)
회 사 명 :
현 장 명 :
작 성 일 : 년 월 일
작 성 자 : (인)
검 토 자 : (인)
현장소장 : (인)
※ 본 양식은 현장별 특성 및 상황에 따라 현장여건에 맞도록 수정하여 사용할 수 있다.
```

### 1.2 본문 표 구조 (p.10-11)

#### 블록 1 — 작성/협의 일자 (2행 × 2열)
```
작 성 일 | 년 월 일
협 의 일 | 년 월 일
```

#### 블록 2 — 회사 현황 (2행 × 4열)
```
회사명 | [값] | 주 소 | [값]
전화번호 | [값] | 현장소장 | [값]
```

#### 블록 3 — 이동식크레인 현황 (3행 × 4열)
```
기종/모델명 | [값] | 성 능 | 톤급
크레인 소유회사명 | [값] | 운전원명 | [값]
운전원 자격·면허 | [값] | 검사유효기간 | [값]
```

#### 블록 4 — 작업개요 반복표 (17행 × 6열, A·B·C일자 3열)
```
구분          | 일시(A) | 일시(B) | 일시(C)
작업 일시 일  | 년월일  | 년월일  | 년월일
작업 일시 시  | 시~분   | 시~분   | 시~분
작업장소       | [값]    | [값]    | [값]
작업내용       | [값]    | [값]    | [값]
관리감독자     | [값]    | [값]    | [값]
작업지휘자     | [값]    | [값]    | [값]
조종원         | [값]    | [값]    | [값]
줄걸이작업자   | [값]    | [값]    | [값]
줄걸이보조자   | [값]    | [값]    | [값]
신호수         | [값]    | [값]    | [값]
신호방법       | □수신호 | □수신호 | □수신호
```

#### 블록 5 — 이동식크레인 능력표 (기계 제원, 다열)
- 이동식크레인 성능, 줄걸이 용구 능력 (와이어로프·슬링), 지형·지반 상태 체크박스, 아웃트리거 설치 방법

#### 블록 6 — 체크리스트
```
항목 | 일시(A) | 일시(B) | 일시(C)
□ 자격증·면허증 휴대 여부
□ 작업방법·작업내용 이해 여부
□ 줄걸이 방법·신호방법 이해 여부
...
```

#### 블록 7 — 작업계획도 (평면도·단면도 영역)

#### 블록 8 — 확인서명
```
상기 작업계획서에 따라 작업을 수행한다.
[확인자 : (서명) (서명) (서명) (서명) (서명)]
```

### 1.3 구조 특징

| 특징 | 내용 |
|------|------|
| 작업 유형 | 이동식 크레인 전용 (기종·성능·줄걸이 상세 포함) |
| 반복 구조 | 날짜 단위 반복 (A/B/C일자) |
| 서명란 | 5인 서명 |
| 도면 영역 | 평면도·단면도 수기 작성 공간 |
| 크레인 특화 항목 | 굴착에는 불필요 — 구조만 참조 |

## 2. 시저형 고소작업대 안전관리계획 (9페이지)

### 2.1 문서 구조

| 섹션 | 내용 |
|------|------|
| 1. 공사현황 | 공사명·금액·기간·소재지·발주자·주요공정, 인력표(성명·직종·경력), 장비 |
| 2. 안전관리조직 | 조직도 (안전보건관리책임자→현장소장→관리감독자→근로자) |
| 3. 안전점검계획 | 표 (구분·실시자·주기·점검항목·비고) + 위험작업 안전대책 (위험요인·안전대책 서술) |
| 4. 현장 비치 서류 | 목록 |
| 5. 안전교육 계획 | 교육 구분·대상·시간·시기·실시자·교육과목 표 |
| 6. 비상조치계획 | 비상연락망 |
| 7. 기타 안전관리 | 서술 |
| 첨부. 위험성평가 | 위험성 추정기준표 + 위험성평가 상세 표 |

### 2.2 위험성평가 표 구조 (참조)

```
공정명 | 세부작업명 | 위험분류 | 위험발생상황및결과 | 관련근거(법적기준) |
현재의안전보건조치 | 평가척도 | 가능성 | 중대성 | 위험성 |
위험성감소대책 | 개선후위험성 | 개선예정일 | 완료일 | 담당자
```

총 15열 — 굴착 작업계획서와 성격이 다름. 위험성평가 표는 별도 문서.

### 2.3 안전점검 반복 표 구조

```
구분 | 실시자 | 주기 | 점검항목 | 비고
순회점검 | 도급인 사업주 | 2일 1회 이상 | [내용] | [비고]
합동안전보건점검 | ... | 2개월 1회 | ... | ...
TBM | ... | 매일 | ... | ...
작업시작전점검 | 관리감독자 | 작업시작전 | ... | ...
```

## 3. KOSHA C-39-2023 §6.1 — 굴착작업계획 법정 항목

직접 표 구조는 없으나 작업계획서에 포함되어야 할 사항을 명시:

| 항목 | 출처 |
|------|------|
| 굴착공법 및 순서, 토사반출 방법 | C-39-2023 §6.1 (가) |
| 소요인원 및 장비 투입 운영계획 | C-39-2023 §6.1 (나) |
| 지하매설물 조사결과에 따른 대책 | C-39-2023 §6.1 (다) |
| 배수처리 계획 (배수장비·경로) | C-39-2023 §6.1 (라) |
| 연락 신호체계 | C-39-2023 §6.1 (마) |
| 계측 계획 (흙막이지보공 설치 시) | C-39-2023 §6.1 (바) |
| 굴착장비별 안전대책 | C-39-2023 §6.1 (사) |
| 재해요인별 안전시설물 설치방법 | C-39-2023 §6.1 (아) |
| 유해가스 측정 및 환기계획 (해당 시) | C-39-2023 §6.1 (자) |
| 복공판 설치계획 (해당 시) | C-39-2023 §6.1 (차) |

|---------|-----------------|----------------------|---------|
| 표지/제목 | ✓ | ✓ | ✓ |
| 메타(사업장명·현장명·작성자·서명) | ✓ | ✓ | ✓ |
| 작업개요 블록 | ✓ | ✓ | ✓ |

## 5. Excel 변환 시 제외 항목

| 항목 | 제외 사유 |
|------|---------|
| 도면 공간 (평면도·단면도) | Excel 셀 내 수기 도면 불가 |
| 체크박스 (□) 항목 | openpyxl 기본 지원 한계 → 텍스트 필드로 대체 |
| 날짜별 반복 열 구조 (A/B/C일자) | 굴착 작업은 1회 작업 단위로 충분 |
| 위험성평가 상세 표 (15열) | 별도 서식 → 이번 builder 범위 외 |
| 안전관리조직도 | 조직도 그래픽 → Excel 범위 외 |
