## 3. 실제 서식과의 주요 차이 (WARN 항목)

### 3.1 법적 근거 부제 누락 (WARN-1)

**실제 MOEL 공식 서식**: 제목 아래 소행으로 법령 근거 표기
```
안전보건교육일지
「산업안전보건법」 제29조에 따른 안전보건교육
```
- 시행규칙 제32조는 교육일지 양식 서식 제4호에서 이 부제를 요구
- 현 builder: 제목 행만 존재, 부제 없음
- 판정: **WARN** — 법령 근거 부제는 공식 서식에서 표준 요소

### 3.2 수강자 행 높이 18pt (WARN-2)

- 18pt ≈ 6.35mm — 손글씨 서명란으로 부적절
- MOEL 표준 서식: 수강자 행 높이 약 8~10mm (23~28pt)
- 최소 권장: 22pt (≈ 7.76mm) — 한글 2줄 입력 가능
- 판정: **WARN** — 인쇄 후 서명 불가 수준

### 3.3 교육 내용 행 높이 20pt (WARN-3)

- 20pt ≈ 7.06mm — 교육 내용 요약 입력 시 1행으로 제한
- 실제 서식: 30~40pt (내용 요약 기록 공간 확보)
- 권장: 30pt (≈ 10.6mm)
- 판정: **WARN** — 내용 기록 공간 협소

### 3.4 확인 서명 행 높이 30pt (WARN-4)

- 30pt ≈ 10.6mm — 수기 서명 공간으로 최소 기준
- 실제 서식: 확인자 서명란 40~50pt (인감·직인 공간 포함)
- 권장: 40pt (≈ 14.1mm)
- 판정: **WARN** — 직인 날인 공간 부족
