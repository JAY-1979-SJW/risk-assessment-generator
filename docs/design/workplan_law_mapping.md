# 작업계획서 법정 요구사항 매핑 — 3단계 (굴착작업)

**작성일**: 2026-04-24  
**법령**: 산업안전보건기준에 관한 규칙 (고용노동부령 제389호)  
**대상 조항**: 제38조 (사전조사 및 작업계획서의 작성 등) + 제82조 (굴착 작업 계획)

---

## 1. 제38조 — 작업계획서 공통 요건

### 1.1 작성 의무

| 조항 | 내용 |
|------|------|
| 제38조 제1항 제6호 | 굴착작업 — 작업계획서 작성 의무 |
| 제38조 제3항 | 작성 완료 후 관계 근로자에게 내용 주지 의무 |

### 1.2 공통 법정 기재 항목 (제38조 제2항)

| 항목 | 조문 | Builder 필드 | 실제 서식 위치 |
|------|------|------------|--------------|
| 작업 방법 | 제38조 제2항 각호 공통 | `work_method` | 법정 항목 블록 |
| 안전보건상 문제 발생 시 조치사항 | 제38조 제2항 각호 공통 | `emergency_measure` | 법정 항목 블록 |

---

## 2. 제82조 — 굴착 작업 법정 기재 항목

### 2.1 법문 전체 (제82조 제1항 각호)

| 호 | 법정 항목 | 법문 원문 |
|----|---------|---------|
| 1 | 굴착의 방법 | "굴착의 방법" |
| 2 | 흙막이 지보공 및 방호망의 종류와 방법 | "흙막이 지보공 및 방호망의 종류와 방법" |
| 3 | 사용하는 기계의 종류 및 능력 | "사용하는 기계의 종류 및 능력" |
| 4 | 토석의 처리 방법 | "토석의 처리 방법" |
| 5 | 용수(湧水)의 처리 방법 | "용수(湧水)가 발생한 경우 용수의 처리 방법" |

### 2.2 항목별 매핑

| 법정 항목 | Builder 필드 | 실제 서식 위치 | 누락 여부 |
|---------|------------|--------------|---------|
| 굴착의 방법 | `excavation_method` | 법정 항목 블록 Row 8 | **없음** |
| 흙막이 지보공 및 방호망의 종류와 방법 | `earth_retaining` | 법정 항목 블록 Row 9 | **없음** |
| 사용하는 기계의 종류 및 능력 | `excavation_machine` | 법정 항목 블록 Row 10 | **없음** |
| 토석의 처리 방법 | `soil_disposal` | 법정 항목 블록 Row 11 | **없음** |
| 용수의 처리 방법 (해당 시) | `water_disposal` | 법정 항목 블록 Row 12 | **없음** |

## 4. 관행 기재 항목 (법정 외)

법령 조문에 명시 없으나 실무 표준으로 통용:

| 관행 항목 | Builder 필드 | 사유 |
|---------|------------|------|
| 사업장명/현장명 | `site_name` | 식별 목적, 실제 서식 전수 포함 |
| 작업 유형 | `work_type` | 제38조 각호 구분 표시 |
| 작업 위치 (상세) | `work_location` | C-39-2023 §4 사전조사 연계 |
| 작업 기간 | `work_date_start`, `work_date_end` | 실제 서식 전수 포함 |
| 작업 책임자 | `supervisor` | C-102-2023 "현장소장" 해당 |
| 작성자 | `prepared_by` | 실제 서식 전수 포함 |
| 작성일 | `prepared_date` | 실제 서식 전수 포함 |

## 7. 중복 항목 검토

- `굴착의 방법` (제82조 제1호) vs `작업 방법` (제38조 공통): 의미상 유사하나 법조 다름. **별도 필드로 분리** (중복 없음)
- `안전보건상의 조치`(제38조) vs `안전조치 반복행`: 반복행은 상세 구현. 공통 항목 `emergency_measure`는 **비상 시 조치** 한정.
