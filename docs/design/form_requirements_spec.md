### 1.4 법령이 명시하지 않은 항목 (임의 추가 금지)

다음 항목은 실무 관행으로 널리 사용되나 해당 조문에 명시 없음:

- 작업 일자·기간 (조문 미명시, 관행 기재)
- 투입 인원 수 (조문 미명시)
- 작업 책임자 성명 (조문 미명시, 단 확인자 서명은 별도 실무 관행)
- 사업장명·현장명 (조문 미명시, 식별용 관행 기재)

→ schema 정의 시 "법정" vs "관행(실무)" 로 명확히 구분한다.

### 3.4 PSM 문서 특성

공정안전보고서는 단일 서식이 아니라 **복수 문서 묶음 (report package)** 이다:
- 각 구성요소별 별도 문서 (도면, 표, 서술문 등)
- 법정 심사 요구 → 표지·목차·심사 정보 등 추가 필요
- Excel 단일 파일로 생성 불가 → 구성요소별 개별 생성 후 패키지

→ 구현 방향: "공정안전보고서 전체"가 아닌 **구성요소별 개별 서식** 으로 분해하여 접근.
