## 2. 공통 추출 구조 (5개 블록)

### 블록 1 — 제목

```
[제목] 굴착 작업계획서
[부제] 「산업안전보건기준에 관한 규칙」 제38조, 제82조에 따른 작업계획서
```

### 블록 2 — 메타 영역 (6개 필드)

| 라벨 | 필드 | 위치 | 관계 |
|------|------|------|------|
| 사업장명/현장명 | `site_name` | 좌측 라벨-값 | [관행] |
| 작업 유형 | `work_type` | 우측 라벨-값 | [관행] |
| 작업 위치 | `work_location` | 전폭 라벨-값 | [관행] |
| 작업 기간 | `work_date_start` + `work_date_end` | 좌측 라벨-값 | [관행] |
| 작업 책임자 | `supervisor` | 우측 라벨-값 | [관행] |
| 작성자 | `prepared_by` | 좌측 라벨-값 | [관행] |
| 작성일 | `prepared_date` | 우측 라벨-값 | [관행/AUTO] |

### 블록 3 — 법정 기재 사항 (7개 필드)

| 라벨 | 필드 | 법적 근거 |
|------|------|---------|
| 굴착의 방법 | `excavation_method` | 제82조 제1항 제1호 [법정] |
| 흙막이 지보공 및 방호망 | `earth_retaining` | 제82조 제1항 제2호 [법정] |
| 사용 기계 종류 및 능력 | `excavation_machine` | 제82조 제1항 제3호 [법정] |
| 토석 처리 방법 | `soil_disposal` | 제82조 제1항 제4호 [법정] |
| 용수 처리 방법 | `water_disposal` | 제82조 제1항 제5호 [법정, 해당 시] |
| 작업 방법 | `work_method` | 제38조 제2항 [법정] |
| 긴급조치 계획 | `emergency_measure` | 제38조 제2항 [법정] |

**라벨 형식**: 전폭 라벨-값 (A열=라벨, B:H열=값, wrap_text=True, 높이 30~40pt)

### 블록 4 — 작업단계별 안전조치 표

반복 테이블 구조 (10행 기본, 입력 수에 따라 확장):

| 열 | 컬럼 범위 | 항목 | 필드 |
|----|---------|------|------|
| 순번 | A (col 1) | 자동 채번 | `step_no` [AUTO] |
| 작업 단계 | B:C (cols 2-3) | 작업 내용 | `step_description` [USER] |
| 위험 요인 | D:F (cols 4-6) | 위험요인 서술 | `hazard_point` [USER] |
| 안전 조치 | G:H (cols 7-8) | 안전조치 서술 | `safety_action` [USER] |

**기본 행 수**: 10행 (최소, 실무 평균 5~8건 작업 단계)

### 블록 5 — 확인 및 서명

3인 서명란:
- 작성자 서명 (`sign_preparer`) — 항상 공란 (수기)
- 검토자/확인자 서명 (`sign_supervisor`) — 항상 공란 (수기)
- 작성일 (`sign_date`) — 입력값 또는 공란

## 3. 제외 항목 목록

| 항목 | 제외 사유 |
|------|---------|
| 도면 공간 (평면도·단면도) | Excel 셀에서 수기 도면 불가 |
| 안전관리 조직도 | 그래픽 → Excel 범위 외 |
| 위험성평가 상세 15열 표 | 별도 서식 (위험성평가표) |
| 체크박스 항목 | openpyxl 기본 체크박스 미지원 → 텍스트 필드로 처리 |
| 비상연락망 조직도 | 그래픽 → emergency_measure 필드로 흡수 |
| 안전교육 계획 | 별도 교육일지 서식으로 처리 |
| 비용·공정관리 항목 | 회사별 커스텀 항목 — 법정 외 |
| 날짜별 반복 열 (A/B/C일자) | 굴착 1회 작업 단위 충분 |

## 4. form_data 구조 확정

```python
{
    # 메타 영역
    "site_name": str | None,          # 사업장명/현장명 [관행]
    "work_type": str | None,          # 작업 유형 (예: 굴착) [관행]
    "work_location": str | None,      # 작업 위치 [관행]
    "work_date_start": str | None,    # 작업 시작일 [관행]
    "work_date_end": str | None,      # 작업 종료일 [관행]
    "supervisor": str | None,         # 작업 책임자 [관행]
    "prepared_by": str | None,        # 작성자 [관행]
    "prepared_date": str | None,      # 작성일 [관행/AUTO]

    # 법정 기재 사항 (제82조)
    "excavation_method": str | None,  # 굴착의 방법 [법정]
    "earth_retaining": str | None,    # 흙막이 지보공 및 방호망 [법정]
    "excavation_machine": str | None, # 사용 기계 종류 및 능력 [법정]
    "soil_disposal": str | None,      # 토석 처리 방법 [법정]
    "water_disposal": str | None,     # 용수 처리 방법 (해당 시) [법정]
    "work_method": str | None,        # 작업 방법 [법정]
    "emergency_measure": str | None,  # 긴급조치 계획 [법정]

    # 안전조치 반복행
    "steps": [                        # 작업단계별 안전조치 [관행]
        {
            "step_description": str | None,  # 작업 단계
            "hazard_point": str | None,      # 위험 요인
            "safety_action": str | None,     # 안전 조치
        }
    ],

    # 서명 영역
    "sign_date": str | None,          # 작성일 (서명란) [관행]
}
```
