# PTW-003 고소작업 허가서 — v1 구현 문서

작성일: 2026-04-25  
버전: v1.0  
상태: DONE

---

## 1. 목적

2m 이상 추락위험 고소작업 수행 전, 법령상 안전조치 이행 여부를 확인·기록하는 현장 실무 표준서식.

---

## 2. 적용 범위

- 작업 높이 2m 이상 추락위험 작업 (원청 요구 기반)
- 비계 위 작업, 이동식 비계 작업, 작업발판 작업, 사다리 작업
- 고소작업대 사용 작업 (스카이차, 붐리프트, 시저리프트)
- 지붕 위 작업, 개구부 주변 작업, 샤프트/피트 주변 작업, 천장 내 작업
- 기타 추락위험이 있는 모든 작업

---

## 3. 법정 별지 서식 여부

**법정 별지 서식 없음.**

산업안전보건기준에 관한 규칙은 고소작업 허가서 서식을 별지로 규정하지 않는다.  
법정 의무는 제42조~제44조의 설비 설치 및 안전조치 의무이며, 허가서 발행 의무가 아니다.

- `legal_status`: `practical`
- 원청 안전 관리 규정에서 허가 요구 시 실무 필수

## 9. 비계/고소작업대/사다리/작업발판 구분

| 서식 | 역할 | 적용 범위 |
|------|------|-----------|
| **PTW-003** | 고소작업 허가 여부 판단 및 작업 전 안전조치 확인 | 모든 고소작업 공통 — 얕은 확인 |
| **CL-001** | 비계·동바리 설치 완료 후 구조 점검 | 비계 전용 — 깊은 점검 |
| **CL-003** | 건설장비 일일 사전점검 | 고소작업대 포함 장비류 전용 |
| **EQ-005** | 고소작업대 사용계획서 | TODO — 향후 구현 |

**사용 원칙:**
- 비계 사용 시: PTW-003(허가) + CL-001(비계 점검) 병행
- 고소작업대 사용 시: PTW-003(허가) + CL-003(장비 점검) 병행
- 사다리·독립형 작업발판: PTW-003 단독으로 충분

## 10. 안전대/추락방지설비 반영 기준

제42조 단계적 의무: ① 작업발판 설치 → ② 추락방호망 설치 → ③ 안전대 착용

- 안전대 착용 시 제44조 부착설비 설치 여부 연동 확인 (섹션 8)
- 안전블록, 수직/수평 구명줄, 추락방호망, 지붕 위 미끄럼 방지 포함

---

## 11. RA-001 위험성평가와의 관계

- RA-001 위험성평가에서 고소작업 관련 위험 식별 → PTW-003 발행 결정
- RA-001 adapter: 고소작업 허가서 관련 허가서로 PTW-003 자동 표시
- adapter 코드는 수정하지 않음 (기존 PTW-003 참조 그대로 유지)

---

## 12. RA-004 TBM과의 관계

- TBM 실시 후 permit_check 항목에 PTW-003 반영
- TBM adapter: 고소작업 허가서 확인 항목으로 PTW-003 표시
- adapter 코드는 수정하지 않음

---

## 13. CL-001 비계 점검표와의 관계

- PTW-003은 허가 단계 — 비계 사용 여부 확인 및 CL-001 병행 안내
- CL-001은 비계 설치 완료 후 구조 전문 점검 (PTW-003 대체 불가)
- builder 내 고정 안내 문구: "비계 작업은 CL-001 비계 설치 점검표와 병행하여 확인한다."

---

## 14. CL-003 장비점검표와의 관계

- PTW-003은 허가 단계 — 고소작업대 사용 확인 및 CL-003 병행 안내
- CL-003은 장비 일일 사전점검 (PTW-003 대체 불가)
- builder 내 고정 안내 문구: "고소작업대 사용 시 장비 점검표 및 장비 상태를 별도로 확인한다."

## 16. 현재 한계

1. **KOSHA PDF 원문 미수집** — K1(P-94-2021), K2(C-74-2015), K3(M-155-2023) PDF 본문 미수집 → NEEDS_VERIFICATION 유지
2. **제37조 풍속 수치 불일치** — builder에 기재된 "풍속 10m/s 기준"은 타워크레인 전용 수치(제37조 ②항). 비계·고소작업대 전용 풍속 기준은 제37조 원문에 없음. builder 수정 금지 지침에 따라 note에만 기록.
3. **EQ-005 미구현** — 고소작업대 사용계획서 별도 구현 필요 (섹션 7 연계 재검토 예정)
4. **비계 세부 구조 기준 (벽이음, 가새 등)** — CL-001 영역으로 PTW-003 미포함
