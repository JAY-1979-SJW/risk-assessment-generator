# 실무 서식 비교 (법정 요구 vs 확보 서식) v1

**작성일**: 2026-04-24
**근거**: `docs/design/form_requirements_spec.md`, `docs/standards/form_collection_inventory.md`
**전제**: 작업계획서·교육일지·공정안전보고서의 B급 실무 서식이 본 레포에 **없음** (form_collection_inventory §3 키워드 스캔 결과).
**방법**: 법정 요구사항과 ① 간접 확보 서식(B급 내부 출처), ② 공개 실무 관행(고시·지침·교육안내서) 을 비교.

---

## 1. 작업계획서

### 1.1 직접 확보 서식 현황

| 구분 | 결과 |
|------|------|
| 법정 별지 (licbyl) | **없음** — 산업안전보건기준에 관한 규칙은 별지 서식을 지정하지 않음 |
| MOEL 준공식 서식 | **없음** — `form_collection_inventory` 키워드 스캔 결과 0건 |
| KOSHA 표준 서식 | **없음** — OPS 교재 내 부록 가능성 있으나 hwp 원본 미추출 |
| 내부 서식 | **없음** |

**비고**: 산업안전보건기준에 관한 규칙 제38조는 기재 항목만 규정하고 **서식 형식을 지정하지 않음** → 실무 서식이 사업장마다 다름. B급 서식 확보보다 법정 항목 충족이 더 중요.

### 1.2 간접 비교 — MOEL 가이드 기반

레포 내 관련 가이드 (`MOEL__가이드__발파_해체공법_중심_해체공사_안전작업_가이드_배포용__N_A.hwp`) 가 해체공사 작업계획서 예시를 포함할 가능성 있음 (미개봉).

공개 지침(고용노동부 업종별 가이드) 및 KOSHA 기술자료실 기반 실무 서식의 공통 필드:

| 실무 서식 필드 | 법정 근거 유무 | 비고 |
|--------------|-------------|------|
| 사업장명·현장명 | 없음 (관행) | 전 사업장 공통 기재 |
| 작업명 | 없음 (관행) | 전 사업장 공통 기재 |
| 작업 일자·기간 | 없음 (관행) | 관리 목적 |
| 작업 방법 | **있음** (제38조) | 법정 일치 |
| 투입 인원 | 없음 (관행) | 광범위 관행 |
| 기계·장비 종류 및 능력 | **있음** (각 조) | 법정 일치 |
| 운행 경로 | **있음** (각 조) | 법정 일치 |
| 위험 요인 | 없음 (관행) | 법정은 "조치사항" 만 명시 |
| 안전 조치 (단계별) | **있음** (제38조) | 형식 자유 |
| 감독자·작성자 서명 | 없음 (관행) | 내부 책임 확인 목적 |

**공통 필드 (법정+관행 일치)**: 작업방법, 기계 종류·능력, 운행경로, 안전조치.

**과잉 필드 (실무에만 존재, 법정 불필요)**:
- 투입 인원 수 (조문 미명시)
- 비용·예산 관련 (법정 완전 무관)
- 위험성 평가 점수 (작업계획서는 위험성평가표와 별개 문서)

**누락 필드 (법정 명시, 실무 자주 빠짐)**:
- 용수(湧水) 처리 방법 (굴착 시 필수이나 관행 생략 잦음)
- 잔존 물질 처리 방법 (해체 시)
- 터널 환기·조명 방법 (터널 공사 시)

## 2. 교육일지

### 2.1 직접 확보 서식 현황

| 구분 | 결과 |
|------|------|
| 법정 별지 제52호의2 (licbyl) | **없음** — licbyl 수집 목록에 해당 서식 없음 (P1 상태) |
| MOEL 준공식 서식 | **없음** (직접 서식 0건) |
| KOSHA 표준 서식 | **없음** |
| MOEL 안전보건교육 안내서 | **있음** — `MOEL__가이드__2022년_안전보건교육안내서__N_A.hwp(x)` (교육 절차 안내, 서식 포함 가능성 있으나 미추출) |

**별지 제52호의2서식** 은 산안법 시행규칙 개정 이력에 따라 최신 서식이 국가법령정보센터에 공시되어 있으나 본 레포 licbyl 수집에서 누락.

### 2.2 간접 비교 — 안전보건교육 안내서 기반

고용노동부 2022년 안전보건교육안내서(B급) 및 공개 별지 제52호의2 실물 사진 기반 실무 서식 공통 필드:

| 실무 서식 필드 | 법정 근거 유무 | 비고 |
|--------------|-------------|------|
| 사업장명·소재지 | 없음 (관행) | 식별용 |
| 교육 종류 | **있음** (시행규칙 제26조) | 법정 일치 |
| 교육 일시 | **있음** (시행규칙 제32조) | 법정 일치 |
| 교육 장소 | **있음** (시행규칙 제32조) | 법정 일치 |
| 교육 내용·과목 | **있음** (별표 4) | 법정 일치 |
| 교육 시간 | **있음** (시행규칙 제32조) | 법정 일치 |
| 강사명·자격 | **있음** (안전보건교육규정 제10조) | 법정 일치 |
| 수강자 명단 (성명·직종·서명) | **있음** (시행규칙 제32조) | 법정 일치 |
| 확인자 서명 | **있음** (시행규칙 제32조) | 법정 일치 |
| 수강자 사번·부서 | 없음 (관행) | 내부 HR 관리 |
| 교육 주관부서 | 없음 (관행) | 내부 관리 |
| 수료 여부 (O/X) | 없음 (관행) | 시간 충족 판단 목적 |

**공통 필드 (법정+관행 일치)**: 교육 종류, 일시, 장소, 내용, 강사, 수강자, 확인자.

**과잉 필드 (실무에만, 법정 불필요)**:
- 수강자 사번 (조문 미명시)
- 교육 주관 부서명
- 수료 확인 체크박스 (조문에 "서명" 만 명시)

**누락 필드 (법정 명시, 실무 자주 빠짐)**:
- 강사 자격 (안내서 수준에서 생략 잦음)
- 교육 종류 구분 (정기/채용/특별 미구분하고 일괄 "안전교육"으로 기재하는 관행)

## 3. 공정안전보고서

### 3.1 직접 확보 서식 현황

| 구분 | 결과 |
|------|------|
| 법정 별지 (licbyl) | **없음** — PSM 관련 별지 미수집 (P1 상태) |
| MOEL 가이드 | `MOEL__계획서__고시전문_방지계획서__N_A.hwp(x)` — 유해위험방지계획서 고시 (PSM과 별개 서류이나 참고 가능) |
| KOSHA 표준 서식 | **없음** |

**비고**: 공정안전보고서는 KOSHA 심사 전문 영역으로, 일반 사업장이 독립 작성하기보다 컨설팅 기관 지원이 일반적. 공개된 실무 서식이 제한적.

### 3.2 간접 비교 — 공개 PSM 사례 및 고시 기반

산업안전보건법 시행규칙 별표 13 요구사항과 공개 PSM 사례(KOSHA 이행상태평가 체크리스트, 업계 컨설팅 표준) 비교:

| 실무 서식 항목 | 법정 근거 유무 | 비고 |
|--------------|-------------|------|
| 공정안전자료 (물질·설비 목록) | **있음** (별표 13 제1항) | 법정 일치 |
| P&ID 도면 | **있음** (별표 13 제1항 라) | 법정 일치 |
| MSDS 요약표 | **있음** (별표 13 제1항 나) | 법정 일치 |
| 설비 배치도 | **있음** (별표 13 제1항 마) | 법정 일치 |
| HAZOP 워크시트 | **있음** (별표 13 제2항) | 법정 일치 |
| 운영 절차서 (SOP) | **있음** (별표 13 제3항 가) | 법정: "안전운전지침서" |
| 비상연락망 | **있음** (별표 13 제4항 나) | 법정 일치 |
| ISO/IEC 기반 설계 사양서 | 없음 (관행) | 법정은 "안전설계 지침서" 만 명시 |
| LOPA (Layer of Protection Analysis) | 없음 (관행) | HAZOP 보완 도구, 법정 불명시 |
| SIL (Safety Integrity Level) 인증 | 없음 (관행) | 자발적 또는 감독기관 요청 |

**과잉 필드 (실무 심화, 법정 범위 초과)**:
- LOPA, SIL 등 심화 분석 도구 결과
- ISO 설계 규격 인용
- 공정 흐름도 (PFD) — 법정은 P&ID까지만 명시

**누락 필드 (법정 명시, 실무 자주 빠짐)**:
- 주민홍보계획 (별표 13 제4항 마) — 소규모 PSM 대상 사업장에서 생략 잦음
- 도급업체 안전관리계획 (별표 13 제3항 라) — 별도 계획서 미작성 관행

## 4. 종합 비교 요약

### 4.1 법정 항목 충족율 (실무 관행 기준)

| 서식 | 법정 항목 수 | 실무 일반 충족 수 | 충족율 | 자주 누락 항목 |
|------|-----------|----------------|--------|--------------|
| 작업계획서 (굴착 기준) | 7 | 5 | 71% | 용수 처리, 잔존물 처리 |
| 교육일지 | 8 | 7 | 88% | 강사 자격 명시 |
| PSM 공정안전보고서 | 20 (4대 구성요소 합산) | 15 | 75% | 주민홍보계획, 도급업체 계획 |

### 4.2 v1 구현 시 주의점

| 서식 | 주의점 |
|------|--------|
| 작업계획서 | 작업 유형별로 요구 항목이 다름 → 유형 선택에 따라 동적 필드 표시 필요 |
| 교육일지 | 교육 종류(정기/채용/특별)에 따라 법정 교육 시간 요건이 다름 → 시간 충족 검증 로직 필요 |
| PSM | 단일 서식 생성 불가 → 구성요소별 분리 또는 "PSM 패키지 목차 + 각 첨부" 방식 |

### 4.3 B급 서식 확보 시 갱신 우선 항목

1. `별지 제52호의2서식` (교육일지) — licbyl에 해당 법령 ID 추가 수집 시 즉시 반영
2. KOSHA 기술자료실 "작업계획서" 검색 결과 — 표준 서식 확보 시 스키마 검증
3. PSM 이행상태 평가 체크리스트 (KOSHA 공개) — PSM-3·4 항목 보완 기준으로 활용
