## 2. 직접 확인 불가 사유

법령(산업안전보건기준에 관한 규칙 제38조)은 차량계 하역운반기계 작업계획서의 **서식(별지/별표)을 지정하지 않는다**.  
따라서 법정 서식 원본이 존재하지 않는 것은 수집 실패가 아닌 법령 구조상의 정상 상태이다.

- 차량계 건설기계 제170조도 마찬가지로 서식 미지정 (기재항목만 열거)
- 고용노동부·KOSHA 공식 예시 양식 별도 수집 없음
