## 6. 조치 권고

### 이미지 기반 PDF (OCR 필요)

| 우선순위 | 방법 | 설명 |
|----------|------|------|
| 1 | pytesseract + pdf2image | PDF → 이미지 → OCR 텍스트 추출 |
| 2 | AWS Textract / Google Document AI | 클라우드 OCR API 활용 |
| 3 | pymupdf (fitz) | pdfplumber 대체, 일부 이미지 PDF 처리 가능 |

### HWP/HWPX 미지원

- hwp5txt (hwp5 패키지) 설치 후 재시도
- 또는 LibreOffice headless 변환: `soffice --headless --convert-to txt`

### 다운로드 실패

- download_status=failed 건 재시도: `python kosha_pipeline.py --retry-failed`
