# 위험성평가표 자동생성기 도움말

## 개요
KRAS 표준 양식 기준 건설현장 위험성평가표를 자동으로 생성하는 프로그램

## 주요 기능
1. 기본정보 입력 (현장명, 공사기간, 발주처 등)
2. 조직구성 관리 (안전관리자, 공정별 담당자)
3. 위험성 추정·결정 기준 설정
4. 작업별 위험요소 자동 검색
5. 빈도×강도 위험성 점수 계산
6. 엑셀 위험성평가표 출력

## 메뉴/탭 구성
- 1탭 기본정보: 현장 기본정보 입력
- 2탭 조직구성: 안전보건 조직 구성
- 3탭 위험성 기준: 가능성/중대성 기준표
- 4탭 위험성평가: 작업별 위험요소 평가
- 5탭 회의 및 서식: 회의록, 서명란

## 사용 방법
1. 기본정보 탭에서 현장정보 입력
2. 조직구성 탭에서 담당자 지정
3. 위험성평가 탭에서 작업내용 입력
4. 자동 검색된 위험요소 확인/수정
5. 엑셀 내보내기로 평가표 출력

## 입력
- 현장 기본정보 (현장명, 공사기간, 발주처)
- 작업내용 (CD배관, 스프링클러 설치 등)

## 출력
- 위험성평가표 Excel 파일 (KRAS 양식)
- 프로젝트 JSON 저장 파일

## 위험성 계산
- 가능성(1~3) × 중대성(1~3) = 위험성(1~9)
- 1~2: 낮음(허용), 3~4: 보통(개선필요), 6: 높음(즉시개선), 9: 매우높음(작업중지)

## 지원 작업 유형
CD배관, 강관배관, 동관배관, 헤드설치, 감지기설치, 수신기설치, 펌프설치, 시운전, 서류작업, 철거작업, 소화기설치

## 사용 예시
"소방공사 착공 전 위험성평가표를 KRAS 양식으로 작성해야 할 때 사용"
